### **Requirements**
- **Python 3.12+**
- `pygame-ce` or `pygame`

### **Installation**
1. Clone the repository:
   ```bash
   git clone https://github.com/niveda1704/pygame.git
   cd pygame/space_shooter
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the game:
   ```bash
   python main.py
   ```
